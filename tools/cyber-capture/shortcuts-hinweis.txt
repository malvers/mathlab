Ohne pynput: macOS „Kurzbefehle“ (Shortcuts)
1. Neuer Kurzbefehl: Aktion „Shellskript ausführen“.
2. Inhalt: /usr/bin/python3 /ABSOLUTER/PFAD/zum/Projekt/tools/cyber-capture/capture.py shoot
3. Speichern, dann Tastatur → Kurzbefehle (oder Dienste) — Tastenkombi zuweisen.

So hast du ein globales Kürzel ohne Python-Daemon. pynput alternativ: pip install -r requirements.txt, dann capture.py listen — ggf. Datenschutz → Eingabeüberwachung für das Terminal/python3.
